Overview & Product Identity

1Design Philosophy

Open Cast is a premium stream control deck designed to resemble luxury cyberpunk control modules. Built for pro broadcasters, the UI combines clean, high-performance typography, radial glows, and glassmorphic cards to create a layout that feels state of the art.

Our key layout choice features dynamic multi-column grids for web/desktop screen surfaces and tabbed panels for mobile displays. High-contrast indicators ensure critical parameters like stream health, fps, and audio levels are legible during intense gameplay.

GoRouter Navigation & Flow Guarding

App Navigation Diagram

Declarative Guards (GoRouter redirect)
If a user profile is missing, the router blocks accesses and redirects to /login.
If the profile is active, but the SharedPreferences flag has_requested_permissions is false, the app automatically intercepts and routes to /permissions.
State Persistency
The setup uses Riverpod's state providers backed by SharedPreferences cache to ensure prompts occur once. Once completed, subsequent app loads direct users straight to /home.

Key UX Systems Specification

Scene Compositor

The studio features an interactive Scene Compositor allowing streamers to click and toggle layouts (PiP, Split Screen, etc.). The layout updates the feed representation on the fly.

WebRTC Streaming

Integrated with WebRTC and RTMP configurations to connect streams directly to platforms (Twitch, YouTube) with optimized audio levels and low latency.

Audio Mixer

Broadcasters can mix different inputs via separate sliders (microphone capture, game soundtrack, background media) directly from the dashboard controls.
